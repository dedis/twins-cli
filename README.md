## `researchercli demo [DOCUMENT_DARC]`

Sets up the demo and requests consent for data whose symmetric key's access is
controlled by `DOCUMENT_DARC`

```
USAGE
  $ researchercli demo [DOCUMENT_DARC]

OPTIONS
  -h, --help       show CLI help
```
